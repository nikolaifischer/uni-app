Wieso wird mir Werbung angezeigt?

Das betreiben dieser App ist mit nicht unerheblichen Kosten für Server und Datenbanken verbunden. Um die Server des LRZ nicht unnötig zu belasten werden die Daten von einem gesonderten Server ausgespielt, der natürlich bezahlt werden muss.

Wieso funktioniert diese App nur für Münchner Universitäten?

Die App hängt von W-LAN Nutzungsdaten ab, die von den meisten Unis zwar erhoben aber nicht veröffentlicht werden. Solltest du Zugriff auf diese Daten einer anderen Hochschule haben, so melde dich bitte per Mail hier: mail@todo.com Ich werde dann versuchen diese Hochschule ebenfalls in die App aufzunehmen.
